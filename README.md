# BEO (Busca Encuentra Ofrece) 🛠️🤝

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black) ![Node.js](https://img.shields.io/badge/node.js-6DA55F?style=for-the-badge&logo=node.js&logoColor=white) ![Firebase](https://img.shields.io/badge/Firebase-039BE5?style=for-the-badge&logo=Firebase&logoColor=white)

**BEO** es una plataforma web integral diseñada para conectar a usuarios con prestadores de servicios técnicos y profesionales. El sistema permite digitalizar la búsqueda de oficios, ofreciendo una comunicación directa y eficiente mediante el uso de tecnologías modernas de renderizado dinámico y bases de datos en tiempo real.

---

## ✨ Características Principales

* **🔍 Motor de Búsqueda:** Módulo avanzado para filtrar profesionales por categorías, oficios y ubicación.
* **📄 Renderizado Dinámico:** Implementación de **EJS** para generar vistas personalizadas de perfiles y catálogos.
* **🔥 Backend con Firebase:** Gestión de autenticación de usuarios, almacenamiento de datos y hosting.
* **💼 Perfiles de Servicio:** Espacio para que los prestadores administren su portafolio, experiencia y contacto.
* **⚡ Comunicación en Tiempo Real:** Actualización dinámica de la oferta y demanda de servicios.
* **📱 Interfaz Multiplataforma:** Diseño responsivo optimizado para dispositivos móviles y escritorio.

## 📂 Estructura del Repositorio

La organización del código fuente se presenta de la siguiente manera:

```text
├── 📁 Root
│   ├── 📄 app.js            # Punto de entrada y configuración del servidor Express
│   ├── 📁 views             # Plantillas dinámicas del lado del servidor (.ejs)
│   ├── 📁 public            # Archivos estáticos servidos directamente
│   │   ├── 📁 css           # Hojas de estilo personalizadas y maquetación
│   │   ├── 📁 js            # Lógica del frontend y configuración de Firebase SDK
│   │   └── 📁 img           # Recursos gráficos, iconos y logotipos
│   ├── 📁 routes            # Definición de rutas y controladores de navegación
│   ├── 📄 package.json      # Definición de dependencias y scripts del proyecto
│   └── 📄 .env              # Variables de entorno (Configuración de Firebase)

```
## 🚀 Instalación y Uso Local

Para visualizar el proyecto en tu entorno local, sigue estos sencillos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/EmilioYair/B-E-O.git](https://github.com/EmilioYair/B-E-O.git)

2. **Instalar dependencias de Node.js**
   Navega a la carpeta y ejecuta:
   ```bash
   npm install
   
3. **Configuracion de FireBase:** - Crea un proyecto en la consola de FireBase.
   * Vincula la aplicacion y obten las credenciales de SDK
   * Configura los servicios de Firestore y Autenticacion
     
4. **Ejecutar el servidor**
   ```bash
   npm start
